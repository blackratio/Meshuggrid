Simple Sass based grid

You don't want to use a complexe grid structure ?

It's time to switch...

Download
Current Development Version: 0.1.0

Quick Installation

Bower or npm

// Bower command
$ bower install meshuggrid

// NPM command for browserify use
$ npm install meshuggrid
If you don't use bower or npm, just download the package

Adding files

// Add this file in your index

<link rel=stylesheet type="text/css" href="styles/meshuggrid.css">


// For Sass preprocessor user :
// Add meshuggrid.scss file
// You can also include file with Sass @import function
@import 'meshuggrid';

Simple to use

HTML

// Add mgrid-row & mgutters class to instantiate grid wrapper
// mColonne-X ( column size ) addition must be egual to colonne number

<div class="mgrid-row mgutters">
   <div class="mColone-1">content<div>
   <div class="mColone-11">content<div>
</div>


// Grid without gutters ?
// Remove mgutters class, simple !

<div class="mgrid-row">
   <div class="mColone-6">content<div>
   <div class="mColone-6">content<div>
</div>

CSS - Standart

// Nothing to do, just add html tag & supported class

SASS - With options

// With sass version, you can configure your grid like you want

// Include first Meshuggrid in Sass file like this
// @ Meshuggrid parameters in order
// ($column-number, $gutters-width, $column-prefix)

// $column-number : Number of column ( default : 12, you can use 16 to )
// $gutters-width : Gutters width in percent ( default : 1% )
// $column-prefix : Change column-prefix class ( default : mColone- )
@include meshuggrid(12, 1%, mColone-);
