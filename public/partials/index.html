Quick & simple to use

Just write html code like this, the result may be 12

<div class="mgrid-row mgutters">
   <div class="mGrid mColone-1">1<div>
   <div class="mGrid mColone-11">11<div>
</div>

If you don't want gutters, just erase "mgutters" class from html like this

<div class="mgrid-row">
<div class="mGrid mColone-6">1<div>
<div class="mGrid mColone-6">11<div>
</div>
